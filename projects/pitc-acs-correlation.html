PITC × ACS Correlations

Methods + assumptions for an interactive tract-level QA and correlation tool comparing PIT count totals to ACS indicators across multiple years (San Diego County, 2013–2025). The interactive map + correlation panel are the second half of this same project.

What this project demonstrates

Geospatial ETL → visualization QA layer

  • Multi-year ingestion using year-configured CSV inputs + shared tract geometry
  • Schema normalization for real-world ACS column inconsistencies
  • Tract-level joins between PIT counts and demographic/socioeconomic indicators
  • Inline QA + interpretability (hover details, legends, correlation summaries)
  • Pearson correlation (r) computation with safeguards for missing data

Data & assumptions

  • Inputs include PIT count totals by census tract and ACS 5-year tract indicators
  • Tract geometry is simplified for performance and interactive use
  • Some PIT years reuse the most recent available ACS vintage
  • Correlation describes linear association only and does not imply causation
