Homeless Encampments & Social Determinants of Health

San Diego County · Census tract level · Privacy-preserving analysis

Project overview

This project analyzes relationships between homeless encampment counts and socioeconomic conditions using American Community Survey (ACS) 5-year estimates at the census tract level.

Encampment data are already aggregated by census tract to ensure privacy and anonymity. No point-level locations are used.

Data sources

  • Encampment counts aggregated by census tract (San Diego County)
  • ACS 5-year tract-level demographic and socioeconomic indicators
  • TIGER/Line census tract boundaries (for reference and QA)

Methods

  • ACS ingestion and variable construction using reproducible R scripts
  • Tract-level joins via GEOID (no spatial point joins)
  • Descriptive statistics and exploratory correlations
  • Regression-oriented modeling workflows for count data

Pipeline structure

  • src/config/ — variable definitions and project settings
  • src/ingest/ — ACS data retrieval and reshaping
  • src/transform/ — tract joins and derived metrics
  • src/analysis/ — descriptives and modeling scripts
